# 四个角色
- 资源拥有者(Resource Owner)
- 客户端(Client)
- 资源服务器(Resource Server)
- 授权服务器(Authorization Server)

# 认证过程三个阶段
- 第一阶段主要是向用户取得授权许可
- 第二阶段主要是申请访问令牌（access_token）
- 第三阶段就是使用access_token获取用户数据。
